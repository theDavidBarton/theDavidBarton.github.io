about me ↝

I've been using the not too meek user name "theDavidBarton" around the web, at the same time I am working as a QA engineer and release manager at French metasearch company liligo.com, where I am using JavaScript (ES6), Node.Js and Google's Puppeteer to automate headless Chrome; and GitLab CI, Kubernetes and Jenkins for deployment and release management.
